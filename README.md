# de-municipal-websites
// A curated dataset of official municipal and city websites in Germany, provided in CSV and JSON formats.

This repository provides a structured overview of official websites of municipalities and cities in Germany. The data is available in both CSV and JSON formats for easy integration into scripts, applications, and data analysis workflows.

The dataset is intended to support research, civic tech projects, and automation tasks such as availability checks, website audits, or comparisons between municipalities.

📂 Contents

* CSV files for simple tabular use and spreadsheet compatibility
* JSON files for programmatic access and API-like usage

🎯 Use cases

* Checking availability and status of municipal websites
* Building directories or search tools
* Analyzing digital infrastructure of local governments
* Supporting transparency and civic tech initiatives

⚠️ Notes

The dataset is maintained manually/semiautomatically and may not always be complete or up to date. Contributions and corrections are welcome.
